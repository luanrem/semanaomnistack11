## :electric_plug: Installation

1. Clone the repo
   ```sh
    git clone https://github.com/luanrem/semanaomnistack11.git
   ```
2. Enter on the folder
  ```sh
    cd semanaomnistack11/<folder>/
   ```
3. Install YARN or NPM packages
   ```sh
   yarn
   // or
   npm install
   ```
4. Run the project
   ```sh
   yarn start
   // or
   npm start
   ```
